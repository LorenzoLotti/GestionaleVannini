# GestionaleVannini

## Per setuppare il progetto:

**(Node.js e npm devono essere necessariamente installati)**

Clona la repository e entraci:

	git clone https://github.com/LorenzoLotti/GestionaleVannini.git
	cd GestionaleVannini

comando Linux/macOS

	./setup.bash

comando Windows

	setup
